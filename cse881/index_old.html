Course Overview

This course is intended for senior undergraduate students who are interested to gain hands-on experience applying computational techniques to solve big data analysis problems. Examples of such problems include: How to detect trending topics in Twitter data? How to predict customers who would churn to a competitor? How to design a large-scale recommender system? Students will learn the basic techniques and programming tools that can be used to provide answers to such questions.

Course Objectives

  • To introduce the basic concepts and methods for big data analysis

  • To teach programming skills and tools for collecting, storing, querying, and analyzing large-scale data

Course Outcomes

  • Students will have a good understanding of the capabilities provided by various data analysis techniques and apply the appropriate ones to solve real-world problems

  • Students will gain hands-on experience on using data analysis tools

  • Students will gain experience designing and developing programs for collecting, querying, and analyzing data

Prerequisite and Background

CSE 331 and CSE 335 and STT 351. Students should demonstrate a basic understanding of algorithms and data structures and methods to create reliable programs. Students must have working knowledge of abstract data types and classes. Such background is needed to ensure that students can understand the sample programs provided in class and able to write their own programs for homework assignments and class project. An integral part of the course is the Hadoop distributed computing platform, which is written in Java programming language. Thus, familiarity with data abstraction, classes, and object-oriented concepts (such as inheritance and polymorphism) will be needed.

Preference Books

There is no main textbook for the class. Lecture notes will be posted online and are based on materials from the following books:

  • Introduction to Data Mining (Pang-Ning Tan, Michael Steinbach, and Vipin Kumar), Addison Wesley, 2006

  • Mining of Massive Datasets (Anand Rajaraman, Jeff Ullman), Cambridge University Press, 2011

  • Hadoop: The Definitive Guide (Tom White), O'Reilly Media, Fourth Edition, 2015

Course Policy

Late submissions will be deducted 25% of the total possible assignment grade (if submitted after the deadline but on the same day) or 50% (if submitted a day after the deadline).
